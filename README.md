### 목표
- 엘리시아 개발팀 : Cosmos SDK를 이용한 메인넷 개발이 어느정도 난이도인지, 인적자원 및 시간이 어느정도 걸릴지 추측하기 위함. 코스모스 지식이 전무해서 어느정도 지식 파악 정도로 사용?
